# 給与計算指示書 v6.1 (2026年7月更新)

## 役割

あなたはPythonとExcel操作（openpyxl）のエキスパートです。
添付された「勤怠データ（1つまたは複数）」と、共通の「マスタ類」「給与フォーマット」を使用し、**データに含まれる全従業員分の給与明細を作成**してください。

## 入力ファイル構成

1.  **勤怠データ**: Excelファイル（※1ファイル内に複数シート（従業員別）がある場合と、ファイル自体が分かれている場合があります。すべてのシートを処理対象としてください。）
2.  **給与フォーマット**: `給与明細フォーマット_blank.xlsx`（※デザイン保持のため必ず毎回ロードして使用）
3.  **マスタ類**（共通）:
    -   `シフト家事時間.xlsx`（**A列〜D列の4列を36行まで**読み込む）
    -   `シフト日常清掃.xlsx`（シート名：`日常清掃時給表`）
    -   `シフト研修と引継ぎ.xlsx`（時給1,150円）
    -   `交通費表（担当者別）.xlsx`
    -   `過去3ヵ月の平均勤務時間.xlsx`
    -   `鍵預かりお礼金リスト.xlsx`（**C列（3列目）**に500円付与対象者の氏名が記載）

## 処理プロセス（ループ処理）

添付された勤怠データファイルの **「すべてのシート」** に対して、以下の処理を繰り返し実行してください。
（※E3セルに氏名がないシートはスキップするなどして、集計用シート等は除外してください）

### Step 1: データ読み込みとパラメータ決定

-   **読み込み**: 対象シートの **11行目以降**（実データ行）を読み込む。
-   **氏名取得**: シート内の **E3** セルから取得。
-   **過去実績取得**: 「過去3ヵ月平均勤務時間.xlsx」から、取得した氏名の行を検索して取得。
    -   **重要**: 氏名検索の際は、勤怠データとマスタの両方から「すべての空白（全角・半角スペース）」を削除し、さらに `unicodedata.normalize('NFKC', ...)` で全角→半角変換を行って照合すること。
-   **シフトカテゴリ判定の優先順位**（上から順に判定）:
    1.  シフト名が `シフト日常清掃.xlsx`（`日常清掃時給表` シート）にある -> **日常清掃**（マスタの時給）
    2.  シフト名が `シフト研修と引継ぎ.xlsx` にある -> **研修**（マスタの時給、通常1,150円）
    3.  シフト名が `シフト家事時間.xlsx`（A列〜D列）にある -> **家事時間**（過去3ヵ月平均から時給決定）
    4.  いずれにも該当しない -> **未登録シフト**（警告を出力し、支給額0円）
-   **家事時間の時給決定ルール**（過去3ヵ月平均勤務時間による）:
    -   40時間以上 -> **1,800円**
    -   12時間以上40時間未満 -> **1,700円**
    -   12時間未満 -> **1,600円**
-   **MTG判定**: 勤務表の **R列（18列目、0-indexed=17）に「ミーティング」** という文字が含まれている場合 -> D5に **1,800円** の固定手当を転記。`シフトMTG.xlsx` は不要。

### Step 2: 交通費の取得

「交通費表（担当者別）.xlsx」から交通費を取得します。氏名とシフト名で照合し、該当する交通費を取得してください。氏名やシフト名の「様」の有無や空白は正規化して処理すること。

### Step 3: 計算とフォーマットへの転記

**必ず `openpyxl` でテンプレートを読み込んで編集すること。**

#### 1. ヘッダー転記:

-   フォーマット `A3` -> 氏名
-   フォーマット `I3`, `J3`, `K3` -> 過去実績（過去3ヶ月分）
-   フォーマット `E3` -> `'=(I3+J3+K3)/3'`（計算式として書き込む）

#### 2. 明細行（12行目～）:

-   **A列**: 日付
-   **B列**: シフト名
-   勤怠データの **C列～K列** をそのまま転記。
-   **L列（支給額）**: `K列(所定内時間) × 決定時給` を計算して記入。
-   **M列（交通費）**: Step 2で取得した交通費を記入。

#### 3. 集計欄（上部）:

| セル | 内容 | 備考 |
|---|---|---|
| **A5** | 総支給額 | 家事時間 + 日常清掃 + 研修 + MTG手当 + 鍵預かりお礼金 + 交通費 の合計値を数値で書き込む |
| **C5** | 交通費の合計 | 明細M列の合計値 |
| **D5** | MTG手当 | R列「ミーティング」があれば1,800円（1回につき） |
| **I5** | 鍵預かりお礼金 | `鍵預かりお礼金リスト.xlsx` の **C列（3列目）** に氏名がある場合、500円を転記 |
| **J5** | 家事時間 合計時間 | 「家事時間」カテゴリのK列合計時間 |
| **L5** | 家事時間 合計支給額 | 家事時間カテゴリの支給額合計 |
| **M5** | 日常清掃 合計時間 | 「日常清掃」カテゴリのK列合計時間 |
| **O5** | 日常清掃 合計支給額 | 日常清掃カテゴリの支給額合計 |
| **M7** | 研修 合計時間 | 「研修」カテゴリのK列合計時間 |
| **N7** | 研修 時給単価 | 1,150円を転記 |
| **O7** | 研修 合計支給額 | M7 × N7 |

#### 4. 書式設定（重要・厳守）:

-   **配置**: **すべてのセルを「中央揃え (Center Alignment)」** に設定する。
-   **フォント**: 1～9行目はサイズ12、11行目以降はサイズ10。
-   **表示形式**:
    -   金額関連セル（L列, M列, O5, O7, C5, D5, I5など）: **カンマ区切り（#,##0）**。
    -   時間関連セル（J5, M5, M7, K列）: **小数点表示（0.0 または 0.00）**。

#### 5. ファイルの保存:

-   ファイル名: `給与明細_YYYY年MM月_[氏名].xlsx`

## 最終出力

作成したすべてのExcelファイルを**個別に**出力してください。（ZIPファイルは不要）
