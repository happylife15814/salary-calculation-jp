# 給与計算スキル（salary-calculation-jp）

このスキルは、家事代行・清掃サービス業向けの複雑な給与計算を自動化し、全従業員分の給与明細Excelファイルを生成するための手順とスクリプトを提供します。

## 概要

毎月アップロードされる「勤務表」と各種「マスターファイル」を照合し、以下のルールに基づいて給与を計算します。

- **家事サービス時給の3段階判定**: 過去3ヵ月の平均勤務時間に基づき、1,600円・1,700円・1,800円のいずれかを適用
- **日常清掃・研修・MTGの時給計算**: 各マスターファイルに登録された時給・固定手当を適用
- **交通費の自動照合**: 担当者（顧客名）ごとの交通費を自動加算
- **鍵預かりお礼金**: 対象者に固定額（500円）を加算

## 必要なファイル

給与計算を実行するには、以下のファイルが作業ディレクトリ（例: `/home/ubuntu/upload/`）に必要です。

1. **【YYYY年MM月】勤務表.xlsx**（毎月更新）
2. **過去3ヵ月の平均勤務時間.xlsx**（毎月更新）
3. **シフト日常清掃.xlsx**（マスター）
4. **シフト家事時間.xlsx**（マスター）
5. **シフト研修と引継ぎ.xlsx**（マスター）
6. **シフトMTG.xlsx**（マスター）
7. **交通費表（担当者別）.xlsx**（マスター）
8. **鍵預かりお礼金リスト.xlsx**（マスター）
9. **給与明細フォーマット_blank.xlsx**（出力用テンプレート）

## 使用方法

### 1. ファイルの準備と確認

ユーザーからファイルがアップロードされたら、まずファイルが揃っているか確認します。
特に勤務表のシフト名が各マスターファイルに登録されているか（未登録シフトがないか）を事前にチェックすることが重要です。

### 2. 給与計算スクリプトの実行

同梱されているPythonスクリプトを使用して給与計算を実行します。

```bash
# スクリプトを作業ディレクトリにコピー
cp /home/ubuntu/skills/salary-calculation-jp/scripts/salary_calculator.py /home/ubuntu/

# スクリプト内のファイルパス（年月など）を必要に応じて修正
# 例: sed -i 's/2026年05月/2026年06月/g' salary_calculator.py

# 実行
python3 salary_calculator.py
```

### 3. 結果の検証

スクリプトは `salary_output_YYYY/` ディレクトリに各従業員の給与明細ファイルを出力します。
出力後、以下の主要セルが正しく転記・計算されているか、数名分をサンプリングして検証してください。

- **A5**: 総支給額（計算式: `=(L33+M33+O5+O7+D5+E5+F5+G5+H5+I5)`）
- **C5**: 交通費合計（計算式: `=M33`）
- **L5**: 家事時間合計支給額
- **O5**: 日常清掃合計支給額
- **O7**: 研修合計支給額
- **D5**: MTG手当
- **I5**: 鍵預かりお礼金
